Using PowerShell New-Item Cmdlet

This blog explains how to use New-Item PowerShell cmdlet to create files and folders within your directory.

Create a new file in the current directory

Copy

New-Item -ItemType File -Name logs.txt -Path .

How this script works?

  • You are making use of New-Item cmdlet to create a new item (the new item in our case is a file)
  • Then you pass in the item type value (File) to -Itemtype parameter
  • Next you pass in the name of the file to the -Name parameter
  • Finally you specify the path using -Path parameter as '.'   '.' denotes current directory you are in.

    • Create a new file in a different directory

    Copy

    New-Item -ItemType File -Name logs.txt -Path d:/

    How this script works?

  • You are making use of New-Item cmdlet to create a new item (the new item in our case is a file)
  • Then you pass in the item type value (File) to -Itemtype parameter
  • Next you pass in the name of the file to the -Name parameter
  • Finally you specify the path (where you want to save the file being created) using -Path parameter

    • Create a new folder in the current directory

    Copy

    New-Item -ItemType Directory -Name logfiles -Path .

    How this script works?

  • You are making use of New-Item cmdlet to create a new item (the new item in our case is a folder)
  • Then you pass in the item type value (Directory) to -Itemtype parameter
  • Next you pass in the name of the folder to the -Name parameter
  • Finally you specify the path using -Path parameter as '.'   '.' denotes current directory you are in.


    • Create a new folder a different directory

    Copy

    New-Item -Itemtype Directory -Name logfiles -Path d:/

    How this script works?

  • You are making use of New-Item cmdlet to create a new item (the new item in our case is a folder)
  • Then you pass in the item type value (Directory) to -Itemtype parameter
  • Next you pass in the name of the folder to the -Name parameter
  • Finally you specify the path (where you want to save the folder being created) using -Path parameter

    • Create a Sub folder

    Copy

    New-Item -Itemtype Directory -Name dailylogs -Path d:/logfiles

    How this script works?

  • You are making use of New-Item cmdlet to create a new item (the new item in our case is a sub-folder)
  • Then you pass in the item type value (Directory) to -Itemtype parameter
  • Next you pass in the name of the folder to the -Name parameter
  • Finally you specify the path of the parent folder (where your sub-folder is going to be saved) using the -Path parameter


    • Create multiple files within a directory

    Copy

    New-Item -ItemType File -Path d:/logfiles/dailylogs/logs.txt, d:/logfiles/weekendlogs/logs.log

    How this script works?

  • You use New-item cmdlet for creating the files
  • Set -Itemtype as File
  • Pass in the paths to your files with a comma separating each individual file's path

    • Note: Since you are naming your files as you pass their location to -Path parameter, you don't have to use the -Name parameter here.


    Create multiple files for multiple folders using wildcard character

    Copy

    New-Item -ItemType File -Path d:/logfiles/* -Name temp.txt

    How this script works?

  • New-Item cmdlet is used for creating the files
  • -ItemType is set as File
  • Then -Path parameter is set as d:/logfiles/*. Basically PowerShell wildcard (*) is used to select every folder that is present within logfiles folder.
  • Name of the file being created is passed to the -Name parameter

    • Note: All sub-folders within the logfiles folder will contain the newly created temp.txt file
